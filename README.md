📊 Car Sales & Profit Analysis Dashboard
📌 Project Overview

This project is a Power BI dashboard that analyzes car sales data to generate meaningful insights about sales performance, profit trends, customer behavior, and booking patterns.

🚀 Key Features
Total Bookings, Sales & Profit Overview
Profit Margin & Acceptance Rate Analysis
Sales & Profit by Car Type (SUV, Sedan, Hatchback, Luxury)
Monthly Sales Trend Visualization
Customer Type Analysis (New vs Returning)
Channel-wise Sales & Profit
Fuel Type Analysis (Diesel vs Electric)
Weekend vs Weekday Booking Insights
🛠️ Tools & Technologies Used
Power BI
Excel / CSV Dataset
DAX (Data Analysis Expressions)
Data Cleaning & Data Modeling
📈 Insights
SUV & Sedan cars generate higher sales
Online channel contributes major revenue
Returning customers have slightly higher contribution
Weekend bookings show different trend compared to weekdays
📂 Dataset
Contains booking details, car type, customer type, sales amount, profit, fuel type, etc.
🙌 Conclusion

This dashboard helps in understanding sales patterns and making data-driven business decisions for car rental/sales companies.
